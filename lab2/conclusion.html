Контрольні питання

1. Для чого використовуються каскадні таблиці стилів (CSS)?

Каскадні таблиці стилів використовуються для оформлення зовнішнього вигляду веб-сторінок, дозволяючи змінювати кольори, шрифти, відступи, розміри і багато іншого.

2. Які методи визначення стилів в HTML-документі ви знаєте?

  • Зовнішні стилі: підключаються через тег <link> до окремого файлу CSS.
    • <link rel="stylesheet" href="styles.css">
  • Внутрішні стилі: використовують тег <style> у самому HTML-документі.
    • <style>
body {
background-color: #f0f0f0;
}
</style>
  • Inline-стилі: прописуються безпосередньо всередині HTML-тегів через атрибут style.
    • <p style="color: red;">Цей текст червоний</p>

3. У чому різниця між принципами каскадування і спадкування?

  • Каскадування: система пріоритетності стилів (inline-стилі мають найвищий пріоритет).
  • Спадкування: стилі батьківських елементів передаються дочірнім (наприклад, шрифт від <body>).

4. Як в HTML-документі оформлюються селектори тегу і клас?

Селектор тегу застосовується до всіх елементів одного типу, а селектор класу застосовується до елементів з певним класом.

/* Селектор тегу */
p {
color: blue;
}

/* Селектор класу */
.my-class {
font-size: 18px;
}

5. Для чого використовується ідентифікатор?

Ідентифікатор id використовується для стилізації або маніпуляції унікальних елементів на сторінці.

#header {
background-color: green;
}

6. Навести приклади оформлення текстових елементів.

  • Колір тексту: 
    p { color: red; }
  • Шрифт і розмір: 
    h1 { font-family: Arial; font-size: 24px; }
  • Жирність тексту: 
    p { font-weight: bold; }
  • Вирівнювання тексту: 
    p { text-align: center; }
  • Підкреслення: 
    a { text-decoration: underline; }
